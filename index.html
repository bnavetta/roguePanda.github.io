Hello, World!

I'm Ben Navetta, and I put this page together for Digital Language Art I. It's based on some things I'd done for another project but never got a chance to finish up.

Here's a bit about the site:

  • Fonts: I used Mr Eaves Sans for body text and M+ 1m for headings. As a programmer who likes tweaking things, I've put a bit too much thought into choosing editor and terminal fonts, hence my love of monospaced programming typefaces.
  • Building: While it's absolute overkill, I set up a Gulp-based build system to compile Sass and run it through PostCSS with autoprefixer and cssnano.
  • Flexboxes: I really really like flexboxes and thought they deserve a bullet point.
